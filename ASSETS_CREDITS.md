# Источники ассетов

Проект использует звуки и графику из fan-приложения О, счастливчик
(KHSM.exe). Это некоммерческая реализация российской версии шоу
Кто хочет стать миллионером.

## Авторы оригинального fan-приложения

- Георгий Лесников - брендбук
- Владимир Орлов, Андрей Азарсков - графика
- Михаил Дексбах, Демид Наумов, Денис Антуфьев, Андрей Ерёмин - тестирование
- Тилль Раш - мобильное приложение для голосования зала

## Условия использования

Из файла Условия.txt оригинала:

Все права на формат шоу принадлежат компании Sony.
Автор fan-приложения не претендует ни на что, программа создана
в развлекательных целях. Любую часть программы можно изменять,
кроме файла KHSM.exe.

Настоящий проект - учебная курсовая работа в НИУ ВШЭ, без
коммерческого использования.

## Что используется

| Файл | Источник | Назначение |
|---|---|---|
| public/sounds/khsm/*.mp3, *.wav | KHSM.zip | оригинальные треки шоу |
| public/images/bg.jpg | graphics/bg_16x10.jpg | фон студии |
| public/images/logo.png | предоставлен отдельно | логотип |
| data/packs/base_1.txt | quest/base_1.txt | стартовый пакет вопросов |

## Своя разработка

Весь код написан с нуля:
- shared - типы и протокол
- server - игровая логика, REST, Socket.IO
- client - три страницы по ролям, компоненты, звуковой менеджер
- тесты, документация
